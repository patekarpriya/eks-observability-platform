# EKS Observability Platform

## 📌 Overview
This project demonstrates a complete observability platform deployed on AWS EKS.

## 🏗 Architecture
- Frontend (UI Application)
- API (Backend Service)
- Kubernetes Manifests
- Monitoring Stack (Prometheus, Grafana)

## 📁 Project Structure
- frontend/ → UI code
- api/ → backend services
- k8s/ → Kubernetes manifests
- monitoring/ → observability tools

## 🚀 Goal
To deploy a scalable, monitored application using AWS EKS with full observability.
